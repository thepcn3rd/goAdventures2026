# Adventures of Learning Go 2026

The following collection of golang programs are for educational use only.  In 2023 I started learning golang and have loved the journey.  Enjoy the simple programs that I build for my home lab, CTFs, and other related activities.

![Waterfall Adams](/picts/waterfallAdams.png)

[Object Analyzer](/objectAnalyzer/README.md) - This project is built to receive the input of IP Addresses, Hashes, or other objects to be evaluated and risk scored.  Then based on the risk and their fidelity determine if they should be regarded as a threat or an object that is trusted.  3 programs exist, an API Server to input the information, a worker to process the information, and the admin server.
